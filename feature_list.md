# Rust Hello World 功能清單

## 模組 1：核心執行環境

| 功能編號 | 功能名稱 | 功能描述 | 優先級 | 備註 |
|---------|---------|---------|--------|------|
| [HW-01] | 程式編譯 | 使用 rustc 編譯器將 Rust 源碼編譯為可執行檔案 | P0 | 需安裝 Rust toolchain |
| [HW-02] | 程式執行 | 運行編譯後的可執行檔案，輸出 "Hello, world!" | P0 | 跨平台支援 |
| [HW-03] | 標準輸出 | 使用 `println!` 巨集將字串輸出到標準輸出 | P0 | 自動換行 |
