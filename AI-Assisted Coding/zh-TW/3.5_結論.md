# 3.5 結論

如果我們在總結本章時問自己 AI 支援對除錯是否有用，我們毫無疑問可以回答是的。我們在這裡看不到像程式碼補全那樣多的附加價值，但對於常見問題的快速幫助（例如第 3.1.3 節中的 CORS 問題）是一項好處。

我們也期望 AI 整合的工具會進一步改進。Microsoft 已經在 Visual Studio 中展示的 GitHub Copilot 整合可能很快就會被納入其他領域和產品中。

AI 支援能再次展示其優勢的領域是您不日常使用或正在學習的程式語言。小錯誤可能導致在 Stack Overflow 和其他論壇上花費數小時的網路研究，而 AI 通常不僅能簡潔地描述問題，還能生成修正後的原始程式碼。
