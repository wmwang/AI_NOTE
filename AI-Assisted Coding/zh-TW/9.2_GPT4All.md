# 9.2 GPT4All

嘗試本機 LLM 最簡單的方式可能是 Nomic AI 的開源程式 GPT4All。該專案的目標是在本機硬體（筆記型電腦或桌面 PC）上執行 LLM，同時提供適合初學者的介面。另一個優點是硬體需求相對較低，應用程式可以在各種硬體上運行。GPT4All 最重要的功能包括以下幾項：

**隱私保護**
GPT4All 非常重視資料隱私。應用程式的模型在本機系統上執行，因此所有輸入資料、處理步驟和輸出都不會離開本機。這將敏感資料被第三方攔截的風險降至最低。

**離線能力**
GPT4All 是一個開源專案，設計為 Windows、macOS 和 Ubuntu 的桌面應用程式。安裝後，您可以在本機安裝 LLM 並在圖形使用者介面（GUI）中使用它們。

**多種模型可用**
GPT4All 不受限於特定的 LLM，而是支援多種模型，如 Llama、Mistral 和 Nous-Hermes。您可以在應用程式介面中輕鬆選擇並使用這些模型。

**使用者友善的介面**
GPT4All 的 UI 設計簡單且直覺，讓您無需額外說明即可直接與其互動。

## 9.2.1 安裝與使用

GPT4All 是一個開源的桌面應用程式，您可以安裝在系統上以與本機安裝的 LLM 互動。您可以從 www.nomic.ai/gpt4all 下載 Windows、macOS 和 Ubuntu 的安裝程式。安裝後，您可以立即啟動應用程式並使用它。

核心功能是聊天，其運作方式類似於商業產品 ChatGPT、Gemini 和 Claude。最大的區別在於，您必須先下載並整合一個模型，然後才能輸入第一個提示。您可以直接在起始頁透過 Find Models 或透過 Models 導航來新增模型。如果您使用一般導航，會先看到本機已安裝模型的列表。點擊 Add Model 按鈕前往模型搜尋並安裝新模型。

**圖 9.1 GPT4All 使用者介面**

GPT4All 為每個模型提供簡短描述和最重要的關鍵指標：

**檔案大小**
使用本機 LLM 時，您的系統硬碟上應該有足夠的可用空間。較小的模型在個位數 GB 範圍內，而 Meta 的 Llama 模型較大版本已需要超過 100 GB 的儲存空間。此儲存空間適用於模型及其權重。較大的模型可以描繪複雜的關係，但也需要更多的儲存空間。

**所需 RAM**
您需要 RAM 來執行模型。模型越龐大，您需要提供的 RAM 越多，以避免速度變慢或當機。

**參數**
參數包括 LLM 的權重和偏差。它們控制模型如何處理語言並產生輸出。LLM 擁有的參數越多，就越能處理複雜問題並產生更好的輸出。高參數數量的缺點是模型會變得更加計算密集和記憶體密集。

**量化（Quant）**
量化表示模型權重的精度。在內部，許多模型使用 32 位元浮點數。精度可以透過量化來降低。常用的值為 4、8 或 16 位元。雖然這種降低會犧牲精度，但會提高速度並減少記憶體需求。

**類型（Type）**
模型類型表示底層模型。可能的類型包括 LLaMA3、Mistral 或 GPT。

一旦您決定了一個或多個模型，就可以下載它們。下載完成後，GPT4All 會在聊天視窗中提供一個下拉式選單，您可以在其中選擇所需的模型，然後將其用於當前的聊天會話。

## 9.2.2 與您的檔案對話

GPT4All 的另一個有趣功能隱藏在 LocalDocs 導航項目背後。GPT4All 允許您整合本機文件並將其用於查詢。由於應用程式完全在本機運行，您的文件不會離開系統，因此是安全的。

若要整合您的文件，您應該透過指定系統上文件所在的目錄來建立一個文件集合。GPT4All 會處理該目錄中的所有文件，然後使其可用於查詢。對於新的聊天，您可以透過 LocalDocs 按鈕選擇要使用的模型和此聊天的文件集合。然後您可以像平常一樣編寫提示，GPT4All 會考慮集合中的文件。如果找到匹配項，您還會收到相關文件的連結。

LocalDocs 功能透過嵌入（embeddings）運作（另見第 12 章）。為此，GPT4All 將文件中的資料以向量形式儲存並記錄其含義。借助這些向量，應用程式能夠非常快速地提供關於文件的問題答案。在內部，GPT4All 使用來自 Nomic AI（GPT4All 的製造商）的嵌入模型。然後，應用程式將找到的文字段落整合到提示中，使輸出更加準確和豐富。

GPT4All 支援純文字、PDF 和 Markdown 作為文件格式。在開發中使用 GPT4All 時，這意味著您可以例如將設計模式或架構的書籍整合為 LocalDocs，或者您可以直接使用程式語言或框架的文件來產生特定的回答。這也讓您可以避免 LLM 通常不知道框架和函式庫最新開發狀態的缺點。

## 9.2.3 結論

GPT4All 是一個開始使用本機語言模型的便利解決方案。它提供了使用者友善的介面，讓您可以新增新模型。除了透過搜尋找到的模型外，您還可以安裝和使用自己的模型。一個非常實用的功能是 LocalDocs，它讓您可以輕鬆地將本機文件整合到提示中，而無需擔心底層機制。

除了這些功能外，GPT4All 還為您提供了透過各種語言綁定存取應用程式的選項，例如您可以在 Python 或 JavaScript 應用程式中存取本機語言模型。

GPT4All 旨在作為入門本機語言模型的工具。運早您會達到應用程式的限制，特別是在專業環境中。在本章的其餘部分，您將了解本機語言模型的 GPT4All 替代方案，這些方案設定和配置稍微複雜一些，但為您提供更好的控制。
