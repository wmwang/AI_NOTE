# 10.3 Groq API

如果您想實驗公開可用的 LLM，但沒有足夠快的電腦，Groq（https://wow.groq.com）是一個非常有幫助的替代方案。這家公司目前為開發者提供免費選項來執行一些精選的語言模型（llama、mixtral、gemma、whisper）。Groq 最引人注目的是回答提示的驚人速度——比使用優質專有硬體快得多，也比 OpenAI API 快得多。這種速度對於實驗來說特別棒！

開發者帳戶受到每分鐘數千個 token 的速率限制（例如，llama-3.1-8b-instant 為每分鐘 20,000 個 token）。然而，由於速度極快，如果您在一個腳本中處理多個提示，這些限制很快就會被超過。當然，我們無法確定這種開發者存取權限能免費多久，因為商業使用始終是需要付費的。

最好為初步實驗設定一個 Python 環境目錄。以下命令適用於 Linux 和 macOS：

```
python3 -m venv groq
cd groq
source bin/activate
pip3 install groq
pip3 install python-dotenv  
```

要使用 Groq，您需要一個 API 金鑰。在 Groq 網站上透過簡單的登入流程（您只需輸入電子郵件地址）後即可獲得此金鑰。將金鑰儲存在專案目錄的 `.env` 檔案中：

```
# File .env
GROQ_API_KEY=gsk_xxx  
```

API 的使用與 OpenAI 或 Ollama 非常相似：

```python
# Sample file groq/hello.py
import os
from groq import Groq
from dotenv import load_dotenv
load_dotenv()
client = Groq(api_key=os.getenv("GROQ_API_KEY"))
completion = client.chat.completions.create(
  model="llama3-8b-8192",
  messages=[
    { "role": "user",
      "content": "How do I use list comprehension in Python?"}])
print("finish_reason:", completion.choices[0].finish_reason)
print("input token:",   completion.usage.prompt_tokens)
print("output token:",  completion.usage.total_tokens)
print()
print(completion.choices[0].message.content)  
```
