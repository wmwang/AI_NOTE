# 3.3 在 Raspberry Pi 上開發

在 Raspberry Pi 等嵌入式裝置上開發軟體時，開發環境通常不是在裝置本身上執行，而是在開發者的工作站電腦上。這導致了略微不同的除錯需求。要將除錯器連接到裝置，必須建立網路連線並進行相應的埠轉發。如果 Raspberry Pi 在沒有網際網路存取的生產系統中執行，良好的應用程式日誌記錄對於之後追蹤錯誤就很重要。

我們將在以下兩節中討論這兩個主題。

## 3.3.1 在 Raspberry Pi 上逐步除錯

經典的除錯涉及在程式碼中設定中斷點並使用除錯器執行程式。一旦到達中斷點，執行就會停止，可以在該點檢查應用程式的狀態。此外，您可以隨後逐步跟隨每個呼叫並檢查變數的內容。

我們尚未為此活動找到 AI 的良好應用。設定中斷點和檢查應用程式狀態仍然是手動工作。然而，AI 可以在設定方面支援我們，使我們能夠開始逐步除錯。我們在 Raspberry Pi 上開發了一個 Python 應用程式，用於控制步進馬達並透過整合的觸控顯示器進行控制。

## 3.3.2 建立日誌條目的協助

當除錯裝置不在現場且可能離線時，將重要訊息寫入日誌檔案是良好的做法。這意味著錯誤也可以在之後被發現。我們參與了一個專案，其中可以從雲端平台配置 Raspberry Pi 電腦上的無線網路，這些電腦透過行動網路連線。Raspberry Pi 上的 Python 應用程式進行相應的系統呼叫以永久儲存設定。

我們嘗試從 GitHub Copilot 獲取建立日誌資訊的支援：

**提示：** Help me debug this function with some logging information.
```python
def set_wifi_wpa_cli(ssid, psk):
  if ssid == '' or psk == '':
    return
  basecmd = ['sudo', 'wpa_cli', '-iwlan0']
  cmd = basecmd + ['list_networks']
  subprocess.run(cmd, capture_output=True)
  out_str = res.stdout.decode('utf-8')
  if re.search(rf"\t{ssid}\t", out_str, re.MULTILINE):
    # network already there
  else:
    cmd = basecmd + ['add_network']
    res = subprocess.run(cmd, capture_output=True)
    netnr = res.stdout.decode('utf-8')
    ...
```

> 要為 set_wifi_wpa_cli 函式新增日誌資訊，您可以使用 Python 內建的 logging 模組。這將幫助您追蹤執行流程並擷取每個命令的輸出。
> ```python
> logging.basicConfig(level=logging.DEBUG,
>   format='%(asctime)s - %(levelname)s - %(message)s')
> ...
> if ssid == '' or psk == '':
>     logging.warning("SSID or PSK is empty.")
>     return
> ...
> if re.search(rf"\t{ssid}\t", out_str, re.MULTILINE):
>     logging.info(f"Network {ssid} already exists.")
> else:
>     cmd = basecmd + ['add_network']
>     logging.debug(f"Running command: {' '.join(cmd)}")
>     res = subprocess.run(cmd, capture_output=True)
>     netnr = res.stdout.decode('utf-8').strip()
>     logging.debug(f"Network number: {netnr}")
> ...
> ```

從回應的摘錄中可以看出（我們在此省略了輸出），Copilot 做得非常好。首先，為 Python 配置了日誌記錄。在進一步的步驟中，日誌輸出根據其重要性進行分析，並相應地用 info、warning 或 debug 標記。這使我們之後更容易過濾日誌檔案。對於系統呼叫（subprocess.run），命令和結果都被記錄下來。AI 還使用了相對現代的 Python F-string 字串表示法。
