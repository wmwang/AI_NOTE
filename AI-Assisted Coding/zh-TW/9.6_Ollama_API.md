# Ollama API

Ollama 提供了一個 RESTful API，可讓您以程式化的方式與語言模型進行互動。預設情況下，該 API 在 `http://localhost:11434` 提供服務，並且可以使用任何 HTTP 客戶端進行存取。

## API 概覽

Ollama API 提供了幾個關鍵端點：

| 端點 | 方法 | 描述 |
|----------|--------|-------------|
| `/api/generate` | POST | 生成回應 |
| `/api/chat` | POST | 聊天補全 |
| `/api/embeddings` | POST | 生成嵌入 |
| `/api/pull` | POST | 下載模型 |
| `/api/push` | POST | 上傳模型 |
| `/api/tags` | GET | 列出本地模型 |
| `/api/show` | POST | 顯示模型資訊 |
| `/api/delete` | DELETE | 刪除模型 |
| `/api/copy` | POST | 複製模型 |
| `/api/ps` | GET | 列出運行中的模型 |

## 生成 API

### 基本生成

```bash
curl http://localhost:11434/api/generate -d '{
  "model": "codellama",
  "prompt": "Write a Python function to reverse a string"
}'
```

### 非串流回應

預設情況下，API 會串流回應。若要獲取單一回應：

```bash
curl http://localhost:11434/api/generate -d '{
  "model": "codellama",
  "prompt": "Write a Python function to reverse a string",
  "stream": false
}'
```

## 使用 Python

### Python HTTP 客戶端

```python
import requests
import json

OLLAMA_BASE = "http://localhost:11434"

def generate_code(prompt: str, model: str = "codellama:13b") -> str:
    """從提示生成程式碼。"""
    response = requests.post(
        f"{OLLAMA_BASE}/api/generate",
        json={
            "model": model,
            "prompt": prompt,
            "stream": False,
            "options": {
                "temperature": 0.7,
                "num_predict": 1024
            }
        }
    )
    return response.json()["response"]

def chat(messages: list, model: str = "codellama:13b") -> str:
    """向模型發送聊天訊息。"""
    response = requests.post(
        f"{OLLAMA_BASE}/api/chat",
        json={
            "model": model,
            "messages": messages,
            "stream": False,
            "options": {
                "temperature": 0.7
            }
        }
    )
    return response.json()["message"]["content"]

# 使用範例
code = generate_code("Write a Python function to calculate fibonacci numbers")
print(code)

messages = [
    {"role": "system", "content": "You are a Python expert"},
    {"role": "user", "content": "Explain list comprehensions"}
]
response = chat(messages)
print(response)
```

### 串流回應

```python
import requests
import json

def stream_generate(prompt: str, model: str = "codellama:13b"):
    """串流生成回應。"""
    response = requests.post(
        f"{OLLAMA_BASE}/api/generate",
        json={
            "model": model,
            "prompt": prompt,
            "stream": True
        },
        stream=True
    )
    
    for line in response.iter_lines():
        if line:
            chunk = json.loads(line)
            if chunk.get("done"):
                break
            yield chunk.get("response", "")

# 使用範例
for text in stream_generate("Write a Python web server"):
    print(text, end="", flush=True)
```

### 串流回應

```javascript
async function* streamGenerate(prompt, model = 'codellama:13b') {
  const response = await fetch(`${OLLAMA_BASE}/api/generate`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      model,
      prompt,
      stream: true
    })
  });

  const reader = response.body.getReader();
  const decoder = new TextDecoder();

  while (true) {
    const { done, value } = await reader.read();
    if (done) break;
    
    const chunk = decoder.decode(value);
    const lines = chunk.split('\n').filter(Boolean);
    
    for (const line of lines) {
      const data = JSON.parse(line);
      if (data.done) return;
      yield data.response;
    }
  }
}

// 使用範例
(async () => {
  for await (const text of streamGenerate('Write a Node.js Express server')) {
    process.stdout.write(text);
  }
})();
```

## API 參數

### 生成參數

| 參數 | 類型 | 預設值 | 描述 |
|-----------|------|---------|-------------|
| `model` | string | 必填 | 要使用的模型 |
| `prompt` | string | 必填 | 輸入提示 |
| `stream` | boolean | true | 串流回應 |
| `context` | array | - | 先前回應中的上下文 |
| `options` | object | - | 模型參數 |

### 選項參數

| 參數 | 類型 | 預設值 | 描述 |
|-----------|------|---------|-------------|
| `temperature` | float | 0.8 | 隨機性（0-2） |
| `top_p` | float | 0.9 | 核取樣 |
| `top_k` | int | 40 | Top-k 取樣 |
| `num_predict` | int | 128 | 最大生成 token 數 |
| `num_ctx` | int | 2048 | 上下文視窗大小 |
| `stop` | array | - | 停止序列 |

### 聊天訊息格式

| 欄位 | 類型 | 描述 |
|-------|------|-------------|
| `role` | string | "system"、"user" 或 "assistant" |
| `content` | string | 訊息內容 |
| `images` | array | 圖像（適用於多模態模型） |

## 效能考量

- **串流**：使用串流回應以獲得更快的首個 token
- **上下文大小**：較小的上下文視窗 = 更快的處理
- **批量處理**：盡可能將多個請求批量處理
- **連接池**：重用 HTTP 連接
- **模型選擇**：選擇適合工作的最小模型

## 錯誤處理

```python
import requests

def safe_generate(prompt: str, model: str = "codellama") -> dict:
    """帶有正確錯誤處理的生成。"""
    try:
        response = requests.post(
            "http://localhost:11434/api/generate",
            json={
                "model": model,
                "prompt": prompt,
                "stream": False
            },
            timeout=60
        )
        response.raise_for_status()
        return response.json()
    except requests.exceptions.ConnectionError:
        return {"error": "無法連接到 Ollama。確保其正在運行。"}
    except requests.exceptions.Timeout:
        return {"error": "請求超時。請嘗試更小的模型或提示。"}
    except requests.exceptions.HTTPError as e:
        return {"error": f"HTTP 錯誤：{e}"}
    except Exception as e:
        return {"error": f"意外錯誤：{e}"}
```

## 總結

Ollama API 為與本地語言模型互動提供了一個清晰且一致的介面。其 RESTful 設計使其可以從任何程式語言存取，而串流支援和可配置參數使得針對 AI 輔助程式設計工作流程進行最佳化變得容易。無論您是構建自訂工具還是整合現有應用程式，該 API 都提供了堅實的基礎。
