# 12.3 載入文件

PDF 格式的年度報告包含有關該計畫的寶貴資訊，可在 WordPress 網站 https://viel-falter.at/ergebnisse/veroeffentlichungen 上找到。為了避免逐一下載 18 份 PDF 文件，我們再次利用了 AI：

**提示：** How do I save linked PDF documents from the website https://viel-falter.at/ergebnisse/veroeffentlichungen to a local folder pdfs using Linux shell commands? Don't use temporary files but Unix pipes and remove duplicates. This site uses only absolute URLs.

我們稍微調整了提示，因為 Claude 3.5 Sonnet 最初建立了臨時檔案，我們認為這不是必要的。除了在第一行建立目錄時多餘的 -p 參數（只有在建立巢狀目錄結構時才需要這個參數）之外，沒有什麼可挑剔的：

```bash
mkdir -p pdfs && \
wget -qO- https://viel-falter.at/ergebnisse/veroeffentlichungen/\
  | grep -oP 'href="[^"]*\.pdf"' | \
    sed 's/href="//;s/"$//' | \
    sort -u | \
    xargs -I {} wget -qnc -P pdfs {}  
```

第 2 行的 wget 命令載入網頁的 HTML 程式碼，透過管線字元轉發給 grep 命令。這裡的 -o 參數用於僅輸出找到正規表示式的行部分。Perl 相容正規表示式（-P）的設定在這種情況下不是必要的，但也不是錯誤。sed 用於從連結中移除前面的 href= 字串和引號。sort -u 移除重複項，最後 xargs 為每個找到的連結執行一個 wget 命令，其中 -P 指定儲存文件的路徑。

如果沒有指出所有 URL 都是絕對路徑，Claude 會在最後一步生成一個帶有前面協議和主機名稱（https://viel-falter.at/）的位址，但在說明文字中會指出這個過程僅適用於相對連結。

這裡使用的命令不一定能在另一個網站上同樣正常運作。grep 命令中的搜尋條件使用了雙引號，這在 HTML 中實際上不是必須的；然而，WordPress 內容管理系統使用了雙引號。

除了 PDF 文件之外，我們還會將一些網站列表納入我們的嵌入模型。這些會在應用程式啟動時從網路下載並轉換為文字。

## 12.3.1 SimpleDirectoryReader 與 SimpleWebPageReader

LlamaIndex 函式庫使用文件（documents）和節點（nodes）作為基本元件。文件是資料的容器，例如 PDF 文件或 Markdown 文件。然而，資料庫查詢的輸出或網頁伺服器的回應也可以是一份文件。文件會儲存元資料（例如檔案的檔案名稱）以及與其他文件的關聯（例如硬碟上的資料夾）。節點是文件的一部分，也包含元資料和關聯。

LlamaIndex 讓我們可以非常輕鬆地將下載的 PDF 文件轉換為內部文件格式：

```python
from llama_index.core import SimpleDirectoryReader
pdf_documents = SimpleDirectoryReader("pdfs").load_data(
  show_progress=True
)  
```

llama_index.core 套件中的 SimpleDirectoryReader 會以 LlamaIndex 文件格式生成 pdfs 資料夾中所有檔案的列表。在我們的案例中，這些是 18 份 PDF 文件。show_progress=True 參數會在命令列中建立進度條。作為第 12.1 節範例的擴展，我們還會使用一些重要的網站作為文件。為此，我們需要 SimpleWebPageReader 函式庫：

```python
from llama_index.readers.web import SimpleWebPageReader
webpages = [
  "https://viel-falter.at/",
  "https://viel-falter.at/monitoring/ziele-vision/",
  "https://viel-falter.at/projektpartner/",
  "https://viel-falter.at/news/",
  "https://viel-falter.at/tagfalter-exkursion/",
];
web_documents = SimpleWebPageReader(
  html_to_text = True).load_data(webpages);  
```

html_to_text 選項指定我們只對網頁的文字內容感興趣。pdf_documents 和 web_documents 現在都包含一份 LlamaIndex 文件列表，我們將在後續步驟中將其轉換為可搜尋的索引。要合併這兩個列表，我們使用 Python 的 + 運算子：

```python
documents = pdf_documents + web_documents  
```

然而，SimpleDirectoryReader 和 SimpleWebPageReader 只是將資料載入 LlamaIndex 的兩個範例。在 LLamaHub（https://llamahub.ai/）網站上，你會找到無數其他格式的 Python 模組。
