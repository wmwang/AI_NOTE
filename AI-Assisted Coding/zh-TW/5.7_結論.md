# 5.7 結論

測試自動化最大的問題在於它是在幕後進行的，與實際的功能開發不同，在產品中是不可見的。因此，應用程式的自動化測試經常被忽視。使用像 ChatGPT 或 GitHub Copilot 這樣的 AI 工具，你可以讓大量工作代為完成，從而在測試實作方面變得顯著更快。這些工具還可以幫助你重構測試以進行最佳化。

如果你沒有撰寫測試的經驗，AI 會幫助你邁出第一步。除了生成的程式碼之外，你還會獲得測試各個部分用途的說明。如果你已經實作過測試並具有工具、架構和設計模式的經驗，你可以將這些經驗融入你的提示中，這將對結果產生正面影響。無論你已經有多少經驗，與手動撰寫測試相比，你都能節省大量的開發時間。
