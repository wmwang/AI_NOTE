# 全自動化 AI SDLC — Workflow 與知識庫閉環

## 基本資訊
- **對象**: 已有六堂課基礎的學員，理解 CLAUDE.md、Agentic Loop、Skill 開發、TDD、Clean Architecture。具備基本 Git/Maven 操作能力。
- **總時數**: 1 天（約 3 小時，含概念講授 + Demo + 實作 + 作業說明）
- **上課時間**: 彈性（可濃縮成 2 小時 workshop 或擴充至半日培訓）
- **教室**: 線上或實體
- **講師**: 待填

## 課程目標
- 能夠執行從 ADO 工單到 PR 的完整 AI 全自駕流程
- 能夠使用 Wiki 查詢與更新，建立知識庫閉環
- 能夠設計與操作 A2P 安全審批流程
- 能夠觸發 AI Code Review 並根據回饋修正違規
- 能夠在真實工單上應用 Agentic TDD

## 每日主題
| Day | 主題 | 核心產出 |
|---|---|---|
| Day 1 | 全自動化 AI SDLC：工單 → 開發 → 測試 → PR → 文件同步 | 完成結業挑戰紀錄表 |

## 共用案例
本課以「訂單 API 連線池耗盡導致 500 錯誤」為貫穿案例，串連 ADO 工單讀取 → Wiki 查詢 → TDD 修復 → CI 測試 → AI Code Review → Wiki 更新 → PR 發送的全部環節。
