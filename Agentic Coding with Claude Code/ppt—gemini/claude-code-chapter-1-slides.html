Fundamentals // 基礎概念

What is Context? (什麼是上下文？)

In the world of LLMs, Context is everything the model "sees" before it generates the next token.

在 LLM 的世界中，上下文是模型在生成下一個標記之前「看到」的所有內容。

"A model is only as smart as the context it is given."

Solution // 解決方案

Agentic Context Management

Claude Code doesn't just "read"; it investigates. (Claude Code 不僅是「讀取」，它還會「調查」。)

terminal // agent_process
➜ Thinking: I need to understand how the user is authenticated.
➜ ls src/auth # Listing auth directory
➜ grep_search "JWT_SECRET" # Searching for secret usage
➜ read_file src/auth/service.ts --lines 1-50 # Reading core logic
➜ Observation: Auth is handled via a custom middleware in line 24.

Tools // 工具集

Tool: grep_search

The High-Precision Radar for your codebase. (代碼庫的高精度雷達。)

  • Search for strings, regex, or patterns.
    搜尋字串、正則表達式或模式。
  • Includes surrounding context lines.
    包含周圍的上下文行數。
  • Automatic filtering of binary files.
    自動過濾二進位文件。
grep_search --help
$ grep_search "handleRequest" --include "*.ts"

// Results:
src/app.ts:24: function handleRequest(req, res) {
src/router.ts:12: router.get("/", handleRequest);

Tools // 工具集

Tool: ls & find

ls // Mapping

Explore directory structures to find relevant folders.
探索目錄結構以尋找相關資料夾。

$ ls src/modules
find // Locating

Search for specific files by name or pattern.
依名稱或模式搜尋特定文件。

$ find "config.json"

"Don't read what you can't find. Don't find what you haven't mapped."

Tool: read_file

Surgical Extraction of information. (信息的「外科手術式」提取。)

Providing Your Own Context

Guide the agent with Project Instruction Files. (使用項目指令文件引導代理。)

  • GEMINI.md: Repo-wide rules.
    整個倉庫的規則。
  • MEMORY.md: Personal/Local notes.
    個人/本地筆記。

    架構概覽。

- Use functional components.

Efficiency // 效率

Filtering Noise (過濾噪點)

Context is a finite resource. Don't waste it on junk. (上下文是有限資源，不要浪費在垃圾上。)

  • .gitignore: Automatically respected.
    自動遵守 .gitignore。
  • Glob Patterns: Narrow your search.
    使用 Glob 模式縮小搜尋範圍。
  • Exclude Flags: Hide irrelevant paths.
    排除無關路徑。
Pro Tip

Use --exclude-pattern to skip massive JSON data or lock files during broad searches.

Best Practices // 最佳實踐

Strategy 1: Specificity

"Be a Sniper, not a Bomber." (做狙擊手，而不是轟炸機。)

Anti-pattern

"Fix the bug in the auth system."

The agent has to search everything from scratch.

Better Way

"Investigate the token expiration in src/auth/jwt.ts."

The agent starts with a direct point of interest.

Best Practices // 最佳實踐

Strategy 2: Empirical Verification

Don't assume. Confirm. (不要假設，要確認。)

Verification Loop
1. Hypothesis: "The bug is in the parser."
2. Action: Create a reproduction script.
3. Observation: Run the script. Does it fail as expected?
4. Conclusion: If yes, you have confirmed context.

Warnings // 注意事項

Common Pitfalls (常見誤區)

  • Ghosting: Searching for symbols that no longer exist.
    搜尋已不存在的符號。
  • Over-reading: Loading 2000 lines when only 50 are needed.
    過度讀取：只需要 50 行時卻加載了 2000 行。
  • Context Drift: Forgetting the original goal while deep-diving.
    上下文漂移：在深入研究時忘記了原始目標。
