The Solution // 解決方案

Model Context Protocol (MCP)

An open standard for connecting AI agents to any data source or tool. (連接 AI 代理到任何數據源或工具的開放標準。)

"Think of MCP as USB-C for AI."

A universal interface that allows different devices to talk to each other seamlessly.

Setup // 設定

Configuring MCP Servers

Register servers in your global or local config. (在全局或本地配置中註冊伺服器。)

// claude_desktop_config.json
"mcpServers": {
  "brave-search": {
    "command": "npx",
    "args": ["-y", "@modelcontextprotocol/server-brave-search"]
  }
}
CLI Shortcut

Use claude mcp add [name] to automatically install and configure popular servers.

Isolation // 隔離性

Project-Level MCP

Avoid "Context Bloat" by only enabling tools where they belong.

通過僅在所屬位置啟用工具來避免「上下文膨脹」。

  • Backend Repo: Enable SQL & Redis MCP.
    後端倉庫：啟用 SQL 與 Redis。
  • Docs Repo: Enable Google Drive MCP.
    文檔倉庫：啟用 Google Drive。
  • Design Repo: Enable Figma MCP.
    設計倉庫：啟用 Figma。

Safety // 安全性

MCP Security & Guardrails

Risks

Unauthorized data access or accidental destructive API calls.

風險：未經授權的數據訪問或意外的破壞性 API 調用。

Best Practices
  • Use Read-Only tokens whenever possible.
    儘可能使用唯讀 Token。
  • Audit server logs regularly.
    定期審計伺服器日誌。
  • Limit project scope for sensitive servers.
    限制敏感伺服器的項目範圍。
