First Steps // 起手式

Command: /init

The foundation of any project. (任何項目的基礎。)

terminal
$ claude /init

What it does:

  • Scans for project structure.
    掃描項目結構。
  • Generates initial CLAUDE.md.
    生成初始 CLAUDE.md。
  • Configures ignore patterns.
    配置忽略模式。

Error Correction // 錯誤修正

Undo & Redo

Did the agent make a mistake? No problem.

/undo

Reverts the last code change and removes the history entry.

撤銷上一次的代碼變更。

/redo

Restores a previously undone change.

重做之前撤銷的變更。

Power Feature // 強大功能

Command: /rewind

The "Time Machine" for your codebase. (代碼庫的「時光機」。)

  • Jump back to any previous state in the session.
    回到會話中的任何先前狀態。
  • Undoes multiple steps at once.
    一次撤銷多個步驟。
  • Perfect for "branching" new ideas.
    非常適合測試新的分支構想。
➜ /rewind

1. Added Navbar (3 min ago)
2. Fixed Auth Bug (8 min ago)
3. Initial Setup (15 min ago)

Efficiency // 效率

Lazy Loading Context

The agent doesn't read everything at once. (代理不會一次讀取所有內容。)

  • Reads summary of memory files first.
    先讀取記憶文件的摘要。
  • Fetches details on-demand.
    按需獲取細節。
The Benefit

Maximizes the available context window for Actual Coding rather than background rules.

為實際編碼保留最大的上下文視窗。

Automation // 自動化

Development Hooks

Automate your workflow with event-driven triggers.

Common Hooks
  • pre-commit: Run linting.
    提交前檢查。
  • post-change: Update docs.
    變更後更新文件。
  • on-success: Notify user.
    成功後通知。
config.json
"hooks": {
  "post-task": "osascript -e 'display notification \"Task Complete!\"'"
}

Customization // 自定義

Custom Skills

Create your own domain-specific slash commands. (建立你自己的領域專屬指令。)

Define a "Skill" with a prompt and a name. The agent will then know how to execute that task reliably every time.

skills/audit.md
# Skill: Audit
Check the current file for
security vulnerabilities and
log them to AUDIT.log.
