Paradigm Shift // 典範轉移

Chat AI vs. Agentic AI

Chat (e.g. ChatGPT)

Micro-Optimization

"Write a function to sort this array."

AI outputs text; user copy-pastes.

Agent (Claude Code)

Macro-Optimization

"Add a login page with JWT auth."

AI plans, creates files, installs packages, and runs tests.

Workflow // 工作流

The Plan-Act-Verify Loop

1. Plan

Analyze requirements and existing code.

分析需求與現有代碼。

2. Act

Execute changes and write files.

執行變更並編寫文件。

3. Verify

Run tests and linting to confirm.

運行測試與代碼檢查以確認。

"Autonomy requires a feedback loop."

Patterns // 設計模式

Spec-Driven Design

"Think before you code."

Create a SPEC.md to define exactly what needs to be built before the agent touches any code.

SPEC.md
# Feature: Navbar
- Must be sticky.
- Show 'Login' if !user.
- Mobile responsive burger menu.

Optimization // 優化技巧

Managing the Window

/clear

Wipe the current conversation history to start fresh.

Use when switching to a completely new task.

/compact

Summarize the previous context to save tokens.

Use during long sessions to keep the agent focused.

Safety // 安全性

Local-First & Safety

Claude Code runs in your environment, not in a cloud sandbox.

  • Full Control: You see every command before it runs.
    完全控制：執行前看到每個指令。
  • Privacy: Code stays local; only context sent to API.
    隱私：代碼保留在本地。
Warning

Always review tool calls that delete or modify large amounts of data.
