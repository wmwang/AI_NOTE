# TSMC Brand Spec for AI Course Slides

## Color Tokens (OKLch)
```css
--bg:      oklch(99% 0.005 250);      /* near-white background */
--surface: oklch(100% 0 0);           /* pure white cards */
--fg:      oklch(15% 0.02 230);       /* nearly black text */
--muted:   oklch(45% 0.015 230);     /* dark grey for secondary */
--border:  oklch(88% 0.008 230);     /* subtle border */
--accent:  oklch(45% 0.18 250);       /* TSMC corporate blue */
```

## Font Stacks
```css
--font-display: 'Roboto', -apple-system, BlinkMacSystemFont, system-ui, sans-serif;
--font-body:    'Roboto', -apple-system, BlinkMacSystemFont, system-ui, sans-serif;
--font-mono:    'Roboto Mono', ui-monospace, 'IBM Plex Mono', Menlo, monospace;
```

## Layout Posture Rules
1. **Hairline borders** (1px, no shadows) — clean, professional appearance
2. **Tabular numerics** with monospace font for data/code
3. **Dense, structured layouts** — information per square inch
4. **One decisive accent** (TSMC blue) used sparingly — CTAs and key highlights only
5. **Sharp corners** (0-4px radii) — professional engineering aesthetic
6. **Section dividers** with subtle border lines
